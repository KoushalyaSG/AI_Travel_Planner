## ✨ Overview

AI Travel Planner is an intelligent travel assistant that creates personalized trip itineraries based on your destination, budget, travel duration, and interests.

Instead of spending hours researching hotels, attractions, and routes, users receive an AI-generated travel plan in seconds.

---

## 🚀 Features

- 🧠 AI-powered itinerary generation
- 📍 Destination recommendations
- 💰 Budget-friendly travel planning
- 🏨 Hotel suggestions
- 🍽️ Restaurant recommendations
- 🎯 Personalized travel preferences
- 🗺️ Day-wise travel schedule
- 📱 Responsive and user-friendly interface
